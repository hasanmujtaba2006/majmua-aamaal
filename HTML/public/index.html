ایپ استعمال کرنے کی ہدایات
🔹 جو شخص ذکر کی مجلس شروع کرنا چاہے، وہ Create Group پر کلک کرے۔
🔹 مجلس کے لیے ایک نام درج کرے اور ایک Password بنائے۔
🔹 یہی شخص امیرِ مجلس ہوگا اور ذکر کا آغاز کرے گا۔

🔹 جو افراد اس مجلس میں شامل ہونا چاہیں، وہ Join Group پر کلک کریں۔
🔹 امیرِ مجلس کی طرف سے دیا گیا Password درج کریں۔
🔹 Password درست ہونے پر آپ مجلس میں شامل ہو جائیں گے اور سب لوگ ایک ساتھ ذکر و اذکار پڑھ سکیں گے۔
App Usage Instructions
🔹 The person who wants to start the Zikr session should tap Create Group.
🔹 Enter a Group Name and create a Password for the session.
🔹 This person will be the Ameer-e-Majlis and will start the Zikr.

🔹 Those who want to join the session should tap Join Group.
🔹 Enter the Password provided by the Ameer-e-Majlis.
🔹 Once the password is verified, you will join the session and everyone can perform Zikr together.
