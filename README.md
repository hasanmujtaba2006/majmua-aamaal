**Majmua Aamaal** is a comprehensive, mobile-first Progressive Web Application (PWA) designed to digitize classical Islamic literature and facilitate global, synchronized recitation (Zikr) sessions. 

Live Demo: [Majmua Aamaal on Render](https://majmua-aamaal.onrender.com)

## 🚀 Key Features

* **Real-Time Global Synchronization:** Engineered using Node.js and Socket.io, allowing multiple users to join secure, live Zikr sessions. The communal counter updates instantly across all connected devices globally.
* **Authentic Digital Library:** Digitized complex, multi-page classical texts including:
  * *Khatme Qadirya Shareef*
  * *Be-Nazeer Amal* (with exact Arabic *eraab* and traditional *Naqsh* grids)
  * *Al-Wazifatul Karima* (featuring complete Surahs and 12+ categorized reading sections)
* **High-Fidelity Typography:** Utilizes custom embedded fonts (`Al Qalam Quran` and `Jameel Noori Nastaleeq`) to ensure 100% Arabic and Urdu phonetic accuracy.
* **Modern UI/UX Architecture:** Designed with a premium "dark glassmorphism" interface, responsive CSS grids, dynamic overlays, and state-driven UI updates for a seamless experience on both mobile and desktop.
* **Session Management & Admin Controls:** Role-based access control allows session creators to navigate the group through different Wazaif, reset progress, and end sessions.
* **PWA Integration:** Configured with a web app manifest and caching strategies, enabling users to install the application directly to their device home screens for a native app feel.

## 💻 Tech Stack

* **Frontend:** HTML5, CSS3 (Glassmorphism UI, Flexbox/Grid), Vanilla JavaScript
* **Backend:** Node.js, Express.js
* **Networking:** Socket.io (WebSocket protocol for low-latency real-time communication)
* **Deployment:** Render
